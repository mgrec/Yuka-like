# Yuka-like

## Description

Yuka like is an mobile app in ReactNative (School Project).

## Components, API etc.

- API : https://world.openfoodfacts.org/
- Components : StyleSheet, Platform, SafeAreaView, Text, View, Button, FlatList, BarCodeScanner & AsyncStorage
